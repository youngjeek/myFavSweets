## 🚀 가상 디저트 가게 소개 웹페이지

1.  **개츠비에 대해 배우기**

    - GraphQL
    - SSR

2.  **사진포스트 업로드 기능**

    - 사진중심 메뉴판

3.  **로그인 기능**

    - 개인 찜목록 페이지
    - 인기상품순 정렬

4.  **배너넣기**

    - 가상의 판매자가 프로모션 배너를 일정기간 메인페이지에 노출
